# 계산기 프로토타입 제안

## 개요
기본 사칙연산과 나머지 연산(MOD)을 지원하는 웹 기반 계산기 프로토타입을 구현합니다.

## 문제 정의
프로젝트.md에서 정의한 계산기의 핵심 기능인 기본 연산을 먼저 프로토타입으로 실현해야 합니다. 이는 나중에 계산 이력, 테마/설정 등의 고급 기능을 추가할 기초가 됩니다.

## 범위 (Scope)

### 포함되는 것
- 기본 UI 구조 (HTML)
- 사칙연산 기능: 덧셈(+), 뺄셈(-), 곱셈(×), 나눗셈(÷)
- 나머지 연산(MOD) 기능
- 숫자 및 연산 버튼 클릭 처리
- 계산 결과 표시
- 초기화(Clear) 기능
- Vanilla JavaScript 기반 계산 로직

### 제외되는 것
- 계산 이력 기능 (향후 추가 예정)
- 테마/설정 기능 (향후 추가 예정)
- 심화 연산 (제곱, 제곱근, 삼각함수 등)
- 반응형 디자인 최적화 (프로토타입 단계)
- 단위 테스트 (프로토타입 단계)

## 주요 기능

### 1. 계산 엔진
- 두 개의 피연산자와 하나의 연산자를 저장하는 간단한 계산 엔진
- 지원 연산자: `+`, `-`, `×`, `÷`, `MOD`
- 0으로 나누기 에러 처리

### 2. 사용자 인터페이스
- 계산 결과/입력값 표시 디스플레이
- 숫자 버튼 (0-9)
- 연산 버튼 (+, -, ×, ÷, MOD)
- 소수점 버튼
- 초기화 버튼 (C)
- 결과 버튼 (=)

### 3. 상태 관리
- 현재 입력 값
- 저장된 값
- 현재 연산자
- 연산 수행 여부 추적
